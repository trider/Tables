Monitoring Deployment



Use this page to upload NDS Monitor deployment data from a .csv file.

To upload terms

  1. Use the Choose Files button to select a csv file.
  2. In the Page Name box, type the name of the NDPedia page that will host the table.
  3. Click Preview to display how the uploaded table will be displayed in NDPedia.
  4. Click Upload to generate the page.

Note

If you upload the table data to an existing page, the table will be appended to the page's existing content.
